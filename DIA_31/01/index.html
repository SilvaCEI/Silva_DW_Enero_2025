onclick => Se activa cuando un usuario hace clic en un elemento HTML. onmouseover => Se produce cuando un usuario mueve el puntero del mouse sobre un elemento. onmouseout => Se produce cuando el puntero del ratón se aleja de un elemento. onkeydown => Se dispara cuando se presiona una tecla. onkeyup => Se produce cuando se suelta una tecla. onchange => Se produce cuando cambia el valor de un elemento HTML. onload => Se produce cuando el navegador termina de cargar la página. onsubmit => Se produce cuando se envía un formulario. onfocus => Se activa cuando un elemento recibe el foco. onblur => Se activa cuando un elemento pierde el foco. onwheel => Ocurre cuando se pasa la rueda del mouse sobre un elemento, también ocurre cuando el usuario se desplaza utilizando un panel táctil. touchmove => Ocurre cuando un usuario mueve un dedo por la pantalla. touchstart => Ocurre cuando un usuario toca un elemento. touchend => Ocurre cuando un usuario quita el dedo de un elemento. onvolumechange => Ocurre cuando se cambia un volumen de medios.
